# Tomb of Horrors BG3 Mod — Developer Workflow

## Overview

```
[BG3 Toolkit]  ──→  edit levels/scenes
      ↓
[Project Folder]  ──→  source files (LSX, assets)
      ↓
[Multitool]  ──→  pack into .pak file
      ↓
[BG3 Mod Manager]  ──→  install mod, set load order
      ↓
[BG3 Game]  ──→  test in-game
```

## Step 0: First-Time Setup

### A. Install the BG3 Toolkit (ONE TIME)
1. Open Steam
2. Right-click **Baldur's Gate 3** → **Properties** → **DLC** tab
3. Enable **"BG3 Toolkit Data"** (free, ~160 GB)
4. Wait for download to complete
5. The Toolkit will appear as a launcher option in Steam

> **Status:** ⚠️ NOT YET DONE — toolkit executable not found on system

### B. Configure the Multitool (ONE TIME)
1. Open `bg3-modders-multitool.exe` (**do not run as Admin**)
2. Go to menu bar → **Configuration**
3. Set **BG3 Installation Directory** → `D:\Steam\steamapps\common\Baldurs Gate 3`
4. **Mods/Profiles Directory** should auto-fill → `%localappdata%\Larian Studios\Baldur's Gate 3`
5. Leave "Unlock Threads" OFF (causes lag)
6. Enable **"Pack .paks to Mod Folder"** for faster testing

### C. Configure BG3 Mod Manager (ONE TIME)
1. Open `BG3ModManager.exe`
2. Point it to your BG3 game directory: `D:\Steam\steamapps\common\Baldurs Gate 3`
3. Point it to your mods folder: `%localappdata%\Larian Studios\Baldur's Gate 3\Mods`

## Step 1: Edit a Level

1. Launch BG3 Toolkit from Steam
2. Open the **TombOfHorrors** project (File → Open Project → `D:\Aaron\development\Tomb of Horrors`)
3. Open the level you want to edit in the Level Editor
4. Make your changes (place tiles, add objects, configure triggers, etc.)
5. **Save** (Ctrl+S)

The Toolkit saves files directly into the project folder as LSX/LSB files under:
`Public\TombOfHorrors\Levels\WLD_TombOfHorrors\`

---

## Step 2: Pack the Mod

### Using Multitool (Recommended — faster)
1. Open Multitool
2. Drag the `TombOfHorrors` project folder onto the Multitool window
3. Click **Pack Mod**
4. If "Pack .paks to Mod Folder" is enabled, the PAK goes straight to your BG3 Mods folder

### Using ExportTool (Manual — more control)
```
ConverterApp.exe create-package "TombOfHorrors.pak" -s "D:\Aaron\development\Tomb of Horrors"
```

---

## Step 3: Install and Test

1. Open **BG3 Mod Manager**
2. Your mod should appear in the left panel (Available Mods)
3. Drag it to the right panel (Active Mods)
4. Click **Export Order to Game**
5. Launch BG3 from Mod Manager (or Steam)
6. Load a save / start new game to test

---

## Step 4: Iterate

- Edit in Toolkit → Pack → Test → Repeat
- For fast iteration: Enable hot-reload in Toolkit (if available) to see changes without repacking
- Use the **Multitool's GameObject Explorer** to look up existing BG3 assets (objects, textures, triggers) you can reuse

## Glossary

| Term | Meaning |
|------|---------|
| PAK | Package file — compressed archive of all mod assets |
| LSX | XML-format Larian data file (human-readable) |
| LSB | Binary-format Larian data file (used in-game) |
| UUID | Unique ID for every mod, object, and asset in BG3 |
| meta.lsx | The mod's manifest — tells BG3 what this mod is |
| WLD_ prefix | Larian naming convention for world/level files |
